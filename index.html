Just The Shortcodes, Please

Plenty of great WordPress plugins focus on providing the Font Awesome library. Font Awesome Shortcodes for WordPress assumes you already know how to include Font Awesome in your theme or site and focuses on giving you a great set of shortcodes to use it with.

This plugin creates a simple, out of the way button just above the WordPress TinyMCE editor (next to the "Add Media" button) which pops up an icon-picker and documentation reference. There are no additional TinyMCE buttons to clutter up your screen, just great, easy to use shortcodes!

Icon Picker and Plugin Documentation

Font Awesome Shortcodes for WordPress features a robust icon-picker for choosing and inserting the icon you need from among Font Awesome's dozens of available options.

The plugin's full documentation is available for reference right from the WordPress editor, so you'll never be stuck hunting the web for that shortcode property you can't remember.

Requirements

Font Awesome Shortcodes for WordPress is designed to work with Font Awesome 4.0 and WordPress 3.7 and requires PHP 5.3+.

Installation

  1. Download and unzip the plugin from the link at the top of this page
  2. Upload the the fontawesome-shortcodes folder to your wp-content/plugins directory.
  3. Activate the plugin in WordPress!
